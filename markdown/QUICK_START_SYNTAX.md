# Quick Start: Finding the Syntax Highlighting Settings

## Where to Find the Settings

### 🎨 Location 1: Sidebar Toolbar (Always Visible)

```
┌─────────────────────────────────────┐
│ Connections          🎨  ➕        │  ← Paintbrush icon in toolbar
├─────────────────────────────────────┤
│  📁 Work Servers                    │
│    • Production Server              │
│    • Staging Server                 │
│                                     │
│  📁 Personal                        │
│    • Home NAS                       │
│    • Raspberry Pi                   │
└─────────────────────────────────────┘
```

**The paintbrush icon (🎨) is in the top-right of the sidebar, next to the + button**

### 📋 Location 2: Plus Menu

```
┌─────────────────────────────────────┐
│ Connections          🎨  [➕]       │  ← Click the + button
├─────────────────────────────────────┤
│                     ┌───────────────┤
│                     │ New Connection│
│                     │ New Folder    │
│                     ├───────────────┤
│                     │ Syntax Highlight │  ← Here!
│                     └───────────────┤
└─────────────────────────────────────┘
```

**Click the + menu and select "Syntax Highlighting"**

### 🖥️ Location 3: Terminal Toolbar (When Connected)

```
┌──────────────────────────────────────────────────────────┐
│ ☑ Sync resize  [Width] [Height] Font: [11pt] 🎨        │  ← Also here
├──────────────────────────────────────────────────────────┤
│  ┌────────────────────┐  ┌────────────────────┐        │
│  │ Terminal Window 1  │  │ Terminal Window 2  │        │
│  │                    │  │                    │        │
└──────────────────────────────────────────────────────────┘
```

**When you have active SSH connections, the paintbrush icon also appears in the terminal controls**

## Steps to Configure

1. **Click** the paintbrush icon 🎨 (in sidebar or + menu)
2. **Add keywords** by typing a word and choosing a color
3. **Press Enter** or click "Add"
4. **Done!** Keywords are saved automatically

## No Active Connections Needed!

Previously, you had to connect to an SSH session to configure syntax highlighting. 

**Now you can configure it anytime:**
- ✅ Before connecting to any server
- ✅ While planning your setup
- ✅ From the sidebar at any time

## Quick Example

1. Click 🎨 in the sidebar
2. Type "error" → Pick red color → Press Enter
3. Type "success" → Pick green color → Press Enter
4. Click "Done"
5. Connect to any SSH server
6. Type commands and watch your keywords light up!

## Keyboard Shortcut Tip

Instead of clicking the + menu every time, you can:
1. **Click the paintbrush icon** directly in the toolbar
2. Or use the **+ menu** if you want to add connections at the same time

The paintbrush button is faster! 🚀
