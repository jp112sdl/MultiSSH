# Quick Start: Drag and Drop Connections

## 🎯 What You Can Do

Move SSH connections between folders using drag and drop!

## 📝 How It Works

### Move Connection to a Folder

```
1. Click and hold on any connection
2. Drag it to a folder header
3. Folder highlights in blue → you're in the right spot!
4. Release to drop
```

### Remove Connection from a Folder

```
1. Drag the connection from its folder
2. Drop it on the top section (where unfoldered connections are)
   OR
   Drop it on "Drop here to remove from folder" zone
```

### Move Between Folders

```
1. Drag a connection from one folder
2. Drop it on another folder's header
3. Done! Connection moved instantly
```

## ✨ Visual Feedback

| What You See | What It Means |
|--------------|---------------|
| 🔵 Blue highlight on folder header | This folder will receive the connection |
| 🔵 Blue highlight on top section | Connection will become unfoldered |
| No highlight | Not a valid drop target |

## 💡 Quick Tips

✅ **Instant save** - Changes are saved immediately, no need to click "Save"
✅ **Visual feedback** - Always watch for the blue highlight to confirm your target
✅ **Undo-friendly** - Just drag it back if you dropped in the wrong place
✅ **Works with collapsed folders** - Drop on folder header even when collapsed
✅ **No accidents** - Dropping outside valid targets does nothing

## 🎬 Example Workflow

**Scenario**: You just added 5 new servers and want to organize them

1. All 5 are in the unfoldered section (top)
2. Drag "prod-server-01" to "Production" folder → drops → ✅
3. Drag "prod-server-02" to "Production" folder → drops → ✅
4. Drag "dev-server-01" to "Development" folder → drops → ✅
5. Drag "staging-web" to "Staging" folder → drops → ✅
6. Keep "backup-server" unfoldered (just leave it at the top)

Done in seconds! No edit dialogs needed.

## 🔄 Alternative Method

If you prefer, you can still organize connections the old way:
1. Click connection to edit
2. Use the "Folder" dropdown
3. Click "Done"

Use whichever method you prefer!

## ❓ Troubleshooting

**Connection didn't move?**
- Make sure you see the blue highlight before releasing
- Try again, dragging directly over the folder name

**Can't find the unfoldered section?**
- If all your connections are in folders, look for "Drop here to remove from folder"
- That's your drop zone for making connections unfoldered

**Folder header not highlighting?**
- Make sure you're dragging a connection (not clicking)
- Try dragging slower and hovering over the folder name

## 🎨 Works Great With

- **Color-coded folders**: See at a glance where connections belong
- **Collapsed folders**: Organize without visual clutter
- **Multiple connections**: Drag them one by one quickly

## 🚀 Pro Tips

⚡ **Fast reorganization**: Drag connections as you think about them, no need to plan ahead
⚡ **Visual organization**: Arrange folders by priority, then drag connections to match
⚡ **Quick triage**: New servers? Drag to "Unsorted" folder first, organize later
⚡ **One-handed operation**: Keep mouse in sidebar area, never need to reach for keyboard
